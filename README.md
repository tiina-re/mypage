# Tiina Reintop Portfolio

A focused React portfolio site built for internship applications. It highlights selected front-end projects, recent certifications, and contact links in a responsive single-page experience.

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- ESLint

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Check code quality:

```bash
npm run lint
```
